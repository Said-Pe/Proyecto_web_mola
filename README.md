# Documentación del Diseño - Molas Guna

## Características Principales

### Diseño Visual
* **Patrón geométrico Guna**: Presente en la sección principal (hero section) y en elementos decorativos.
* **Gradientes coloridos**: Inspirados en los colores tradicionales de las molas.
* **Animaciones suaves**: Transiciones y efectos hover para una experiencia fluida.
* **Tarjetas de producto mejoradas**: Con un borde superior colorido que aparece al pasar el cursor.
* **Badges de descuento**: Estilo moderno con gradientes para resaltar ofertas.

### Funcionalidades JavaScript
**main.js incluye:**
* Animaciones de scroll automáticas
* Lazy loading de imágenes
* Filtros de productos interactivos
* Búsqueda en tiempo real
* Controles de cantidad con botones +/-
* Sistema de notificaciones (toast)
* Validación de formularios
* Prevención de doble submit
* Contador de caracteres en textareas
* Manejo de errores en imágenes

**carrito.js incluye:**
* Agregar al carrito con AJAX
* Actualizar cantidad sin recargar la página
* Eliminar productos con diálogo de confirmación
* Animaciones al agregar/eliminar productos
* Notificaciones visuales de acciones
* Actualización automática de totales

## Diseño Responsive
El diseño está optimizado para:
* **Móviles** (< 768px)
* **Tablets** (768px - 1024px)
* **Desktop** (> 1024px)
